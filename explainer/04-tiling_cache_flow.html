The core idea: tiling
A 512 MB weight matrix never sits in the cache all at once. The GPU slices it into small tiles — chunks that fit in L1 SRAM — processes each tile, evicts it, then loads the next. The cache is a window that slides across the full matrix, not a container that holds it.
